Sequential Search Implementation

Sequential Search is a method which sequentially checks each element of the list until a match is found or the whole list has been searched.
Below are the functions that were implemented along with the procedure of implementation.
No of Occurences function:-  To find the no of occurences, traverse the array and increment each time the key is matched with the elements of the array.
First Occurence function:-  Traverse the array from left to right and return the index position of the first occurence of the key in the array of the elements.
Last Occurence function:-  Traverse the array from right to left and return the index position of the first occurence of the key in the array of the elements.
